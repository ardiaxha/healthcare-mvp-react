# Healthcare Frontend

This repository contains the frontend implementation for the Healthcare MVP.

# Overview

A simple React application to display patients and their appointments. Features include:

- Patient list with search and selection
- Detailed patient view with medical history
- Scrollable, responsive appointments panel
- Modern, clean, and responsive UI
- Handles long text gracefully with scrollable panels and truncated appointments

# Installation

git clone https://github.com/ardiaxha/healthcare-mvp-react.git
cd healthcare-mvp-react
npm install
npm run dev
